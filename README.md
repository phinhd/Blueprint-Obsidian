# Blueprint Obsidian

This folder contains the project documentation organized as an Obsidian vault.

## Get Started

For the best experience, open this folder in **Obsidian**.

The entry point for all documentation is:
**[[Project_Index]]** (found in `00_Meta/Project_Index.md`)

## Structure

- `00_Meta/`: Navigation, Glossary, Attachments.
- `01_Context/`: User Personas, Business Rules, Project Goals.
- `02_Architecture/`: Database, Diagrams, ADRs.
- `03_API_Docs/`: API Endpoints with JSON examples.
- `04_Workflows/`: Business logic flows with Mermaid diagrams.
- `05_Development/`: Setup, Conventions, Handoffs.
